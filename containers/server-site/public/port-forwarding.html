üîå Port Forwarding & Security

Configure your router to allow external connections

Required Ports

To allow others to connect to your server, you need to open the following ports:

Required Ports:
  • 25565 (TCP) - Minecraft server (required for players to connect)
  • 24454 (UDP) - Voice chat mod (required for voice chat functionality)
  • 7681 (TCP) - Web terminal (optional, for server management)
  • 80 (TCP) - Player guide site (optional, for this website)

How to Port Forward

Port forwarding allows external devices to connect to your server through your router. The exact steps vary by router manufacturer, but the general process is:

  1. Access your router's admin panel (usually at 192.168.1.1 or 192.168.0.1)
  2. Log in with your router credentials
  3. Navigate to "Port Forwarding" or "Virtual Server" settings
  4. Add a new port forwarding rule for each port:
    • Port 25565 (TCP): Forward to your server's local IP address
    • Port 24454 (UDP): Forward to your server's local IP address
  5. Save the settings and restart your router if needed
üí° Finding Your Server's Local IP: On Windows, run ipconfig in Command Prompt. On Linux/Mac, run ifconfig or ip addr. Look for your local network IP (usually starts with 192.168.x.x or 10.x.x.x).

Security Recommendations

Security Best Practices:
  • Only open port 25565 if you want others to connect
  • Consider using a firewall to restrict access to specific IPs
  • Port 7681 (web terminal) should only be opened if you need remote access
  • Port 80 (this site) is optional and can be closed if not needed
  • Voice chat port 24454 is required for voice chat functionality
  • Use strong passwords for your server and router admin panel
  • Consider using a whitelist on your Minecraft server to control who can join
  • Keep your router firmware updated

Alternative: Using Ngrok

If you can't configure port forwarding on your router, or want a quick way to share your server, consider using Ngrok. Ngrok creates a secure tunnel to your server without requiring router configuration.
