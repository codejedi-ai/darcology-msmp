üöÄ Getting Started

Your complete guide to joining the server

Step 1: Choose a Launcher

You can use any of these popular Minecraft launchers:

Prism Launcher

Modern, open-source launcher with great mod support.

Download Prism Launcher

MultiMC

Lightweight launcher with excellent instance management.

Download MultiMC

ManyMC

Fork of MultiMC with additional features.

Download ManyMC

Step 2: Install Minecraft Forge 1.20.1

  1. Open your chosen launcher
  2. Create a new instance
  3. Select Minecraft 1.20.1
  4. Install Forge version 47.4.0 or compatible

Step 3: Install Server Mods

  1. Download the mods zip file using the link in the sidebar
  2. Extract all .jar files from the zip
  3. Copy all mod files to your launcher's mods folder:
    • Prism Launcher: Right-click instance ‚Üí Open Folder ‚Üí mods/
    • MultiMC/ManyMC: Right-click instance ‚Üí Open Folder ‚Üí mods/
  4. Make sure all mods are installed before launching

Step 5: Connect to the Server

  1. Launch Minecraft with your modded instance
  2. Click Multiplayer
  3. Click Add Server
  4. Enter server address: localhost:25565 (or use the address provided by the server admin)
  5. Click Done
  6. Select the server and click Join Server
üí° Tip: If the server is using Ngrok or a custom domain, the server admin should provide you with the correct address. It will look like example.ngrok.io:12345 or minecraft.example.com:25565.
