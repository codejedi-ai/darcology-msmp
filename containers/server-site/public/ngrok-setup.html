üåê Using Ngrok for Public Access

Expose your server without router configuration

What is Ngrok?

Ngrok allows you to expose your local server to the internet without port forwarding. This is useful if you can't configure your router or want a quick way to share your server.

Step 1: Install Ngrok

Download Ngrok from https://ngrok.com/download and install it on your computer.

Step 2: Create an Ngrok Account (Free)

  1. Sign up at https://dashboard.ngrok.com/signup
  2. Get your authtoken from the dashboard
  3. Run: ngrok config add-authtoken YOUR_TOKEN

Step 3: Expose Minecraft Server Port and Voice Port

Both ports are compulsory! Open two terminals and run:

# Terminal 1: Minecraft server (REQUIRED)
ngrok tcp 25565

# Terminal 2: Voice chat (REQUIRED)
ngrok udp 24454

Ngrok will display forwarding addresses like:

# Minecraft server
tcp://0.tcp.ngrok.io:12345

# Voice chat
udp://0.udp.ngrok.io:54321

This is your public server address! Share the hostname and port with players:

  • Minecraft Hostname: 0.tcp.ngrok.io
  • Minecraft Port: 12345
  • Voice Hostname: 0.udp.ngrok.io
  • Voice Port: 54321

Step 4: Share with Players

Players should connect using:

Server Address: 0.tcp.ngrok.io:12345
(Replace with your actual Ngrok hostname and port)

Important Notes

  • Free tier limitations: The Ngrok URL changes every time you restart Ngrok. For a permanent address, consider the paid plan or use port forwarding.
  • Keep Ngrok running: Both tunnels (Minecraft and voice chat) must stay active for players to connect. Don't close the terminals running Ngrok.
  • Security: Anyone with the Ngrok URL can connect. Consider using a whitelist on your Minecraft server.
  • Performance: Ngrok adds some latency. For best performance, use direct port forwarding instead.

Optional: Expose Web Terminal

If you want to expose the web terminal, open an additional terminal:

# Terminal 3: Web terminal (optional)
ngrok http 7681

Note: The Minecraft server (port 25565) and voice chat (port 24454) are already required in Step 3 above.
