Common issues and solutions

Can't Connect to Server

  • Make sure the server is running
  • Verify you have all the required mods installed
  • Check that you're using Minecraft Forge 1.20.1
  • Ensure mod versions match between client and server
  • Verify the server address is correct (check the home page for the current address)
  • If using Ngrok, make sure both tunnels (Minecraft and voice) are running
  • Check your firewall settings - it may be blocking the connection
  • Try connecting from a different network to rule out local network issues

Mod Compatibility Issues

  • Download the exact mods from the server (use the Download Mods link in the sidebar)
  • Remove any client-side only mods that aren't on the server
  • Check that all mods are for Minecraft 1.20.1
  • Verify Forge version compatibility (47.4.0 recommended)
  • Make sure you're not using incompatible mod versions
  • Try removing all mods and reinstalling them one by one to identify problematic mods
  • Check the server logs for mod-related errors

Launcher Issues

  • Make sure you've completed the offline bypass steps if using it (see Getting Started)
  • Don't delete the "No Profile" account in Prism Launcher
  • Try creating a fresh instance if problems persist
  • Update your launcher to the latest version
  • Clear the launcher cache and try again
  • Check that Java is properly installed and up to date

Performance Issues

  • Allocate more RAM to Minecraft in your launcher settings (4GB minimum recommended)
  • Close other applications to free up system resources
  • Update your graphics drivers
  • Lower your video settings in Minecraft
  • Check if your computer meets the minimum system requirements

Voice Chat Not Working

  • Verify that the voice chat port (24454 UDP) is properly forwarded or exposed via Ngrok
  • Check that both Minecraft and voice ports are accessible (both are required)
  • Ensure your voice chat mod is installed and matches the server version
  • Check your microphone permissions in your operating system
  • Verify that voice chat is enabled in the mod settings

Still Having Issues?

If you're still experiencing problems after trying these solutions:

  • Check the Getting Started guide to ensure you've completed all setup steps
  • Review the Port Forwarding or Ngrok Setup guides if you're hosting a server
  • Contact the server administrator for assistance
  • Check the server logs for error messages
